Source changelog
- Merged February security patch (android-13.0.0_r30)
- Allow to hide status bar battery icon
- Allow to silent notification when screen is on
- Allow to turn off screenshot feedback

- Ported edge touch optimization from OOS 12 kernel for games (Stock kernel only)

Note
